# lazy-html-plugin

## Dev

```bash
yarn workspace @tarik02/lazy-html-plugin watch
yarn workspace @tarik02/lazy-html-plugin client:watch
```

## Before publish

```bash
yarn workspace @tarik02/lazy-html-plugin build
yarn workspace @tarik02/lazy-html-plugin client:build
```

### Test an example

```bash
yarn watch
yarn serve
yarn build
```
